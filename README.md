# AI Navigation for Visually Impaired Individuals

An AI-powered navigation assistance system designed to support visually impaired individuals through real-time object detection and audio guidance using YOLOv5.

---

## Overview

This project detects surrounding objects through computer vision and provides guidance that can assist users in understanding their environment more effectively.

The system combines:

* Real-time object detection
* Camera input processing
* Audio-based navigation assistance
* YOLOv5 detection architecture

---

## Features

✔ Real-time object detection
✔ Audio assistance for navigation
✔ Camera-based environment awareness
✔ Obstacle identification
✔ Efficient detection pipeline

## Installation

```bash
git clone https://github.com/husnajuman5/AI-Navigation-For-Visually-Impaired-Individuals.git
cd AI-Navigation-For-Visually-Impaired-Individuals
pip install -r requirements.txt
```

---

## Run

```bash
python detect.py
```

or

```bash
python audio_guidance.py
```
